Overview Install Bookmarklet

Bookmarklet Installer — Phase 1, Version 1.0.0

This page installs the AMA11Y bookmarklet into your browser. The bookmarklet audits any web page for accessibility issues entirely within your browser. It makes zero outbound network calls and works in any environment including secured corporate networks and virtual desktops accessed via VPN.

Private instrument for Akhilesh Malani only. This tool is not for public distribution.

Step 1: Copy the Bookmarklet Code

The full bookmarklet code is in the text area below. Press the Copy to Clipboard button. JAWS will announce Copied to clipboard when it succeeds.

This text area contains the complete bookmarklet. It begins with the word javascript followed by a colon. Do not edit it. Use the button below to copy it.

Step 2: Add to Bookmarks Bar in Your Browser

After copying, follow the instructions for your browser below. Use the Tab key to move between browser tabs and press Enter or Space to select one.

Installing in Google Chrome

  1. First show the bookmarks bar if it is hidden. Press Control + Shift + B. JAWS will not announce this change, but the bar will now be visible on screen.

  2. Open the Bookmark Manager by pressing Control + Shift + O. A new tab opens titled Bookmark Manager.

  3. Press Tab until you reach the Add new bookmark button and press Enter. JAWS announces a dialogue.

  4. In the Name field, type: AMA11Y Audit

  5. Press Tab to move to the URL field. Press Control + A to select any existing text, then press Control + V to paste the bookmarklet code you copied in Step 1.

  6. Press Tab until you reach the Save button and press Enter.

  7. The bookmarklet is now saved. By default Chrome saves to Bookmarks Bar. If asked for a folder, choose Bookmarks Bar.

  8. Close the Bookmark Manager tab by pressing Control + W.

Alternative method in Chrome: Press Control + D on any page to open the Bookmark dialogue. Change the Name to AMA11Y Audit. Then Tab to the URL field, clear it, and paste the bookmarklet code. Save to Bookmarks Bar.

Installing in Microsoft Edge

  1. Show the favourites bar if hidden. Press Control + Shift + B. The favourites bar will appear below the address bar.

  2. Open the Favourites Manager by pressing Control + Shift + O. A panel opens on the right side of the screen. JAWS announces Favourites.

  3. Press Tab to navigate within the panel. Look for a button labelled Add favourite or the plus icon button and press Enter.

  4. In the Name field, type: AMA11Y Audit

  5. Tab to the URL or Folder field. If a URL field is present, press Control + A then Control + V to paste the bookmarklet code.

  6. Ensure the folder is set to Favourites Bar. Press the Save or Done button.

Edge alternative using the address bar: Navigate to any page. Press Alt + D to focus the address bar. Press Control + D to open Add to Favourites. JAWS announces a dialogue. Change the name to AMA11Y Audit, change the folder to Favourites Bar, then find the URL field, paste the bookmarklet code, and press Save.

Installing in Mozilla Firefox

  1. Show the bookmarks toolbar if hidden. Press Alt to open the menu bar. Arrow right to View, press Enter. Arrow down to Toolbars, press Enter. Arrow down to Bookmarks Toolbar and press Enter to enable it.

  2. Open the Library window by pressing Control + Shift + B. JAWS announces Library.

  3. Tab to the Bookmarks Toolbar item in the left panel and press Enter to expand it.

  4. Press Application key or Shift + F10 to open the context menu. Arrow down to Add Bookmark and press Enter. A dialogue opens.

  5. In the Name field, type: AMA11Y Audit

  6. Tab to the Location field. Press Control + A then Control + V to paste the bookmarklet code.

  7. Tab to Save and press Enter.

Step 3: Verify the Installation

  1. Navigate to any web page. For a first test, use a simple page such as bbc.co.uk or gov.uk.

  2. To reach the bookmarks bar with JAWS in Chrome or Edge, press Alt + D to focus the address bar, then press Shift + Tab once. JAWS should announce the bookmarks or favourites toolbar. Use the left and right arrow keys to find AMA11Y Audit and press Enter.

  3. In Firefox, press Alt to reach the menu bar, then press B for Bookmarks, arrow down to Bookmarks Toolbar, arrow right to find AMA11Y Audit, and press Enter.

  4. Wait a moment. JAWS should announce:
    AMA11Y audit complete. [number] failures found.
    Then press Insert + F5 in JAWS to refresh the virtual buffer if the panel does not receive focus automatically.

  5. Press H to navigate to the AMA11Y Audit Results heading. Press T to jump to the findings table. Use the arrow keys to read each finding.

Keyboard Reference for the Audit Panel

Key Action
EscapeClose the AMA11Y panel and return to the page
H (JAWS virtual cursor)Navigate between headings in the panel
T (JAWS virtual cursor)Jump to the findings table
Arrow keys in tableMove between rows and cells. Each cell announces its column header.

If the Audit Does Not Run on a Page

Some pages use a Content Security Policy that blocks browser scripts from running. If you activate the bookmarklet and nothing happens, the page is blocking it. This is not a fault in AMA11Y. It is a security feature of the page. Note the page URL and report it so we can log it. The Phase 2 browser extension bypasses this restriction entirely.
