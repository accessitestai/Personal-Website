Document Accessibility Audit

Upload a PDF or Word document to audit its accessibility structure. PDF audit checks tag structure, language, alt text, headings, tables, and bookmarks. DOCX audit checks heading styles, alt text, table headers, language, lists, reading order, bookmarks, and colour contrast of text.

Upload Word Document for Accessibility Audit

Checks: Heading styles, alt text, table headers, language, lists, text boxes, bookmarks, colour contrast

Note: Both audits use structural and content analysis. Scanned PDFs with no machine-readable text will report no tag structure. Password-protected documents cannot be audited.

How AI powers AMA11Y v5.0.0

  • GPT-4o — Describes exactly what JAWS or NVDA announces for each failing element. Also generates copy-paste code fixes.
  • Claude — Writes the full professional audit report: executive summary, priority roadmap, and impact narratives from the blind user's perspective.
  • Gemini — Explains any WCAG criterion in depth: what it requires, how screen reader users are affected, and the exact test procedure.
