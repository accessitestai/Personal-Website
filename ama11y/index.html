AMA11Y

Accessibility Management and Audit Layer

A blind-first accessibility audit tool that runs entirely in your browser. Built for screen reader users. Zero network calls. Works behind firewalls and on locked-down corporate desktops.

Install the Bookmarklet

What It Audits

AMA11Y runs 8 accessibility audit engines against any web page, covering WCAG 2.2 Level AA and AAA criteria. Every finding includes the element reference, criterion, computed value, required value, verdict, severity, and remediation guidance — all announced by screen readers.

Focus Order

SC 2.4.3, SC 2.1.2 — Tab sequence, positive tabindex detection, focus trap analysis

Focus Visibility

SC 2.4.7, SC 2.4.11, SC 2.4.12 — Indicator suppression, outline contrast ratio

Colour Contrast

SC 1.4.3, SC 1.4.6, SC 1.4.11 — Text AA/AAA ratios, non-text UI component contrast

Heading Structure

SC 1.3.1, SC 2.4.6 — Single H1 validation, skipped levels, empty headings

Landmark Regions

SC 1.3.1, SC 2.4.1 — Required landmarks, duplicate detection, unique naming

Image Alt Text

SC 1.1.1 — Missing alt, generic alt detection, SVG accessible names, decorative image review

Form Labels

SC 1.3.1, SC 3.3.2 — Programmatic label associations, required field indicators

Links and Buttons

SC 2.4.4, SC 4.1.2 — Generic link text detection, missing accessible names

How It Works

  1. Install the bookmarklet

    Copy the bookmarklet code and add it to your browser favourites bar. One-time setup, works in Chrome, Edge, and Firefox.

  2. Activate on any page

    Navigate to the page you want to audit, move to your favourites bar with keyboard, and press Enter on the AMA11Y bookmark. The audit runs instantly.

  3. Review findings in the accessible panel

    Results appear in a fully accessible panel. Navigate by headings, filter by verdict or engine, expand finding details, and export reports in JSON, HTML, CSV, or plain text.

Arrow keysNavigate between table rows and cells

TabMove between interactive controls (filters, export buttons, close)
Insert + F5Refresh JAWS virtual buffer if content not announced

Technical Details

Zero network calls. The bookmarklet runs entirely within the browser tab. No data is transmitted anywhere. Works behind firewalls, on VPNs, and on locked-down corporate desktops.

  • Browser support: Chrome, Edge, Firefox (any version supporting ES6)
  • Screen reader support: JAWS 2024+, NVDA 2023+, VoiceOver (macOS Sonoma+)
  • Braille display support: Compatible with Freedom Scientific Focus 40 Blue
  • No installation required: Single bookmark — no extensions, no IT approval, no admin rights
  • Export formats: JSON, HTML report, CSV (for Excel), plain text
  • Standards coverage: WCAG 2.2 Level AA and AAA criteria
