Before you send

For a bug, the most useful thing you can include is: which page you were on, what you tried to do, what happened instead, and whether you were using the Chrome extension or the platform. Screenshots are welcome but not required. If you are blind or use a screen reader, please say which one so I can reproduce.

Name and email are optional. If you leave them blank the message is anonymous and I cannot reply. If you include an email I will only use it to respond to this specific message.

All fields marked optional can be left blank. Message is the only required field.
