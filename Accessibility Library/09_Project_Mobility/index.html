AI-Powered Mobility Assistant

For Wheelchair Users & Visually Impaired Individuals

Authors: Akhilesh Malani and L. Subramani

Date: June 24, 2025

Project Overview

Comprehensive development of a flexible, AI-powered mobile solution designed to significantly enhance:

  • Mobility independence
  • Quality of life
  • Accessibility for individuals living with disabilities

A unified platform serving both wheelchair users and visually impaired individuals

Target Users

Wheelchair Users

  • Need accessible route planning
  • Require barrier identification
  • Benefit from optimized navigation

Visually Impaired Individuals

  • Need real-time obstacle detection
  • Require audio feedback
  • Benefit from voice guidance

Addressing critical mobility challenges with innovative AI solutions

Key Features Overview

Wheelchair Navigation

AI-powered route optimization

Obstacle Detection

Real-time environmental awareness

Voice Guidance

Intuitive audio interface

Customizable Interface

Personalized user experience

Wheelchair Navigation System

AI-powered route planning algorithms that generate optimized routes

Actively Identifies and Circumvents:

  • Staircases and steep paths
  • Narrow passages
  • Uneven surfaces
  • Physical barriers

Ensuring safe, accessible routes for wheelchair users

Obstacle Detection System

Real-time, low-latency detection using smartphone camera

Provides Clear Audio Feedback:

  • Distance: How far obstacles are
  • Direction: Where obstacles are located
  • Hazard Type: What kind of obstacle

Converting visual information into actionable audio guidance

Voice Guidance Interface

Robust and intuitive voice interface

Features:

  • Turn-by-turn directions
  • Immediate hazard alerts
  • Natural voice synthesis
  • Low latency responses

Adaptability:

  • Wheelchair navigation mode
  • Visually impaired mode
  • Multiple language support
  • Customizable voice settings

Technical Architecture

Platform

Android (initial) with future iOS expansion

AI Models

  • OpenCV: Computer vision processing
  • TensorFlow Lite: Machine learning optimization
  • MobileNet/SSD: Lightweight object detection

APIs & Services

  • Google Maps API: Navigation and mapping
  • OpenStreetMap: Accessibility data
  • Text-to-Speech: Natural voice interface
